# Contributing

PRs welcome. Before opening a PR:

1. Run `./gradlew detekt ktlintCheck testDebugUnitTest`
2. Match existing code style; prefer Compose primitives over custom views
3. New features start as an issue on [Linear](https://linear.app/cursor-android-app/project/cursor-agents-android-99ff64c53382). If your change is bigger than a one-liner, open an issue first.
